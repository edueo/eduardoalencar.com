## Requisitos

## Instalando Hugo

**Via homebrew no macOS/Linux**: `brew install hugo`

## Executar o site localmente

1. Clonar esse repositório
```
git clone git@github.com:edueo/eduardoalencar.com.git
```

2. Executar o comando
```
make dev
```

3. Voilá
http://localhost:1313

## Deploy

```
make deploy
```
